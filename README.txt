
# BAPBAP Site avec Connexion Steam

## ✅ Installation rapide

1. Installe Node.js : https://nodejs.org
2. Ouvre un terminal dans ce dossier.
3. Installe les dépendances :
   ```
   npm install express express-session passport passport-steam dotenv
   ```
4. Ajoute ta clé Steam dans `.env` :
   https://steamcommunity.com/dev/apikey

5. Lance le serveur :
   ```
   node server.js
   ```
6. Ouvre : http://localhost:3000
